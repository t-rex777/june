# June

A Social Media App.

Live Link : https://june-social.netlify.app/

## Tech Stack
 **Frontend**
 * React
 * Typescript
 * Redux Toolkit
 * Tailwind CSS
 
 **Backend**
 * NodeJs
 * Express
 * MongoDB
 * Cloudinary to store photos
 
 ## Key features
 
 * JWT authorization
 * create posts
 * like, comment, follow and unfollow
 * Edit profile picture and user details
 * Search using debouncing
 * user suggestions
 * notifications
